# App Assets

Place your app icons and splash screens here:

## Required Files

1. **icon.png** (1024x1024)
   - App icon for both iOS and Android
   - Should be a square PNG with no transparency

2. **splash.png** (1284x2778)
   - Splash screen shown while app loads
   - Recommended size for iPhone 13/14 Pro Max

3. **adaptive-icon.png** (1024x1024)
   - Android adaptive icon foreground
   - Should have transparent background

4. **favicon.png** (48x48)
   - Web favicon if using Expo web

## Generating Assets

You can use these tools to generate all required sizes:
- https://easyappicon.com/
- https://appicon.co/
- Expo's asset generator: `npx expo-asset-gen`

## Placeholder

Until you have custom assets, you can:
1. Create simple colored squares as placeholders
2. Use the Expo default assets
3. Generate from your logo using the tools above
